# 🧠 Dynamic Quiz Web Application

An elegant, highly interactive, and responsive single-page quiz portal designed to evaluate users through dynamic query loading. Built completely from scratch featuring dynamic state triggers and smooth layout transitions.

---

## 🛠️ Tech Stack Used

* **HTML5:** Semantic interface architecture wrappers (`.cont`, `.quiz`, `.ans-btn`).
* **CSS3 Engine:** External font imports (`Poppins`), deep spatial dual-color background gradients, responsive layouts (`flex`), and interactive color mapping.
* **Vanilla JavaScript (ES6+):** Dynamic Node Creation (`createElement`), Event Delegation listeners (`mouseover`/`click`), Local DOM Manipulation node loops, and custom element dataset mutations (`dataset.correct`).

---

## ✨ Key Features Included

* **Interactive State Validation:** Built-in dynamic state systems automatically attach custom visual classes (`.correct` / `.incorrect`) depending on chosen targets.
* **Smart Interaction Shield:** Answers freeze automatically via conditional disabled locks (`button.disabled = true`) once a choice is locked in, blocking double-clicks.
* **Asynchronous Progress Controls:** The application dynamically displays structural iteration triggers (`.next-btn`) only after active response capture.
* **Score Evaluation Matrix:** Real-time state computation modules calculate absolute player metric metrics and display raw outcomes dynamically upon quiz termination.
* **Garbage Collection Loops:** Active system cleanup nodes (`removeChild`) purge previous button nodes from browser execution paths before rendering new queries.
